# Examples

All examples expect `OPENAI_API_KEY` to be set:

```bash
composer install
export OPENAI_API_KEY="..."
```

Run:

```bash
php examples/01-text-response.php
php examples/02-conversation.php
php examples/03-structured-json.php
php examples/04-tool-calling-manual.php
php examples/05-tool-calling-conversation.php
php examples/06-websearch.php
php examples/07-text-to-speech.php
php examples/08-custom-attachment.php
```
